   - Monthly Household Income
   - Household Size
   - 4Ps Beneficiary (Yes/No)
   - PWD in Household (Yes/No)
   - Senior Citizen in Household (Yes/No)
   - Solo Parent (Yes/No)
   - Disaster-Affected (Yes/No)
